Colors and animations

Link to website: Check this out

In this project we learned how to apply a color palette to our design and more importantly apply the right proportions, for primary, secondary and accent colors.

We've learned how to apply a theme color and switch between dark and light mode using CSS and JS.

Finally we added animations to the graphical elements using JS .animate function and the keyframes and properties for animating objects.

Implementing a color pelette

While primary and secondary colors occupy most of the space on an page, the accent colors are meant to attract the viewer's attention. They are usually a call to action and/or the UI elements. Accent colors are more vibrant and are meant to stand out from the background in contrast.

Adding animations

Just like adding animations in CSS, animations can be added with JS. While properties and keyframes can be defined as objects, the selected DOM elements can be animated by adding the animate. function.
